Hurricane Mapping Project

This project was completed to see if there was a correlation between rising sea surface temperatures and the frequency/strength of storms in the Atlantic. On this page you will find the maps I created, a GIT link for the actual code, and the final report. The maps are interactive and feature a hover function. All data obtained is sourced in the Final Project overview.
